Carla Gabriela Hernández Nuñez developer

Soy Carla Gabriela Hernández Nuñez, desarrolladora apasionada por crear soluciones funcionales y visualmente atractivas. Egresada de la Universidad Politécnica Metropolitana de Hidalgo, he participado en proyectos web y de software aplicando tecnologías como .NET, JavaScript, PHP y más. Me enfoco en seguir aprendiendo, colaborar en equipo y construir herramientas que generen impacto real.

Experiencia

Aplicación web para servicios de comida

Desarrollo de un sistema web completo para gestionar pedidos de comida en una colonia, con funcionalidades como carrito de compras, seguimiento de pedidos, roles de usuario y control de acceso. El objetivo fue impulsar las ventas de pequeños negocios locales.

Sistema de control de acceso a laboratorios

Durante mis prácticas profesionales desarrollé un sistema en PHP para controlar el acceso a laboratorios mediante lectores de huellas dactilares. La información se almacenaba en una base de datos, y se creó una aplicación web para consultar horarios de ingreso de estudiantes, profesores y encargados.

Sistema para Agencia Aduanal

Participación en el desarrollo de un sistema interno para una agencia aduanal. Se implementó un CRUD completo, análisis de archivos para extracción de datos, y generación de reportes personalizados para facilitar la gestión documental.
